# HWE Bench — site

Static site for [HWE Bench](../README.md). Generated by `python -m tools.site.build`
from `bench/results.jsonl` and per-rep artifacts.

## Layout

```
site/
├── index.html              Home — hero, leaderboard, the "unbounded" thesis
├── methodology.html        How fitness is measured + the three correctness gates
├── models.html             Per-model deep dive — every rep, every winning hypothesis
├── data.html               Index of raw data files (log.jsonl, agent.log, summary.json)
└── css/
    ├── tokens.css          Design tokens (color / type / spacing)
    ├── reset.css           Modern CSS reset
    ├── style.css           Global styles + components
    └── print.css           Print stylesheet (clean PDF output)
```

The site has zero JavaScript. Fonts load from Google Fonts.

## Build

```sh
python -m tools.site.build
```

Reads `bench/results.jsonl` and each rep's `log.jsonl` / `summary.json`,
re-renders the four HTML pages, leaves CSS / fonts untouched.

A GitHub Action (`.github/workflows/pages.yml`) re-runs the build and
deploys to GitHub Pages on every push to `main` that touches the site
or the bench data.

## Design system

See `.design/branding/hwe-bench/`:

- `BRIEF.md` — brand brief
- `strategy/STRATEGY.md` — positioning, voice, messaging
- `identity/IDENTITY.md` + `identity/palettes.json` — color, type, spacing tokens
- `patterns/STYLE.md` — composition rules, layout, tables, print stylesheet
